 **Important Note:** This is the **Front End** version of the application. Backend development is currently in progress. The complete application with full backend integration will be uploaded soon. Stay tuned for updates.
 
 ## About The Project
 Kushhal Kisan is a comprehensive mobile application designed specifically for **farmers and livestock owners** in **Balochistan**. The app serves as a bridge between the agricultural community and government officials, enabling seamless communication, problem reporting, and expert guidance.

### Target Audience
| Category | Description |
|----------|-------------|
| **Farmers** | Crop farmers, agricultural workers |
| **Livestock Owners** | Cattle, goat, sheep, camel, and poultry owners |
| **Government Officials** | Agricultural extension officers |
| **Rural Communities** | People involved in agriculture and livestock |

### Primary Focus: **Balochistan**

The app is tailored to address the unique agricultural and livestock challenges faced by the people of Balochistan, including:
- Water scarcity issues
- Drought management
- Livestock diseases
- Crop selection for arid regions
- Market access problems
- Communication gap between farmers and government officials.

  ## Key Features

### **Farmer-Official Communication**
| Feature | Description |
|---------|-------------|
| **Problem Reporting** | Farmers can report agricultural/livestock issues directly to officials |
| **Direct Messaging** | One-on-one chat with government officers |
| **Photo/Video Upload** | Share visual evidence of crop diseases or livestock problems |
| **Location Sharing** | Pinpoint exact farm/location for site visits |
| **Real-time Responses** | Get immediate replies from officials |
| **Issue Tracking** | Track status of reported problems |

### **Farmer Features**
| Feature | Description |
|---------|-------------|
| **Weather Updates** | Real-time weather forecasts for Balochistan |
| **Crop Guidance** | Crop selection, planting, harvesting tips |
| **Market Prices** | Current market rates for crops and livestock |
| **Government Alerts** | Subsidy schemes, weather warnings, disease outbreaks |
| **Educational Videos** | Tutorials in local languages (Urdu, Pashto, Balochi, Brahui) |
| **Tools Information** | Agricultural machinery and equipment guides |

###  **Livestock Features**
| Feature | Description |
|---------|-------------|
| **Cattle Care** | Feeding, breeding, healthcare for cows/buffaloes |
| **Goat/Sheep Management** | Specialized guides for small ruminants |
| **Camel Farming** | Camel health, breeding, and management |
| **Poultry Farming** | Chicken, duck, and other poultry guides |
| **Disease Identification** | Common livestock diseases with symptoms and treatments |
| **Vaccination Schedule** | Timely vaccination reminders |
| **Vet Connect** | Connect with veterinary officers |

### **Government Official Features**
| Feature | Description |
|---------|-------------|
| **Secure Login** | Service ID-based authentication |
| **Dashboard** | View all farmer requests and complaints |
| **Issue Resolution** | Respond to farmer problems with solutions |
| **Tips & Techniques** | Share agricultural/livestock best practices |
| **Analytics** | Track common issues in different regions |
| **Area Mapping** | Identify problem areas for targeted support |

## Installation Guide
### Prerequisites
- Flutter SDK (3.35.4 or higher)
- Android Studio / VS Code
- Android SDK (for Android development)

### Steps to Run

1. **Clone the repository**
2. ## Get dependencies
3. flutter pub get
4. ## Run the app
5. flutter run
6. ## Run on Specific Platform
7. # Android (Mobile)
flutter run -d android
# Web Browser
flutter run -d chrome
# Windows Desktop
flutter run -d windows
